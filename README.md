# 🐹 Learning Go (Go Playground)

A personal workspace and laboratory where I document my journey of learning Go (Golang) from scratch. This repository contains my structured study notes, syntax experiments, and hands-on mini-projects.

## 🎯 Primary Focus: Gophercises

My immediate goal is to work through the challenges from **[Gophercises](https://gophercises.com/)** by Jon Calhoun. By building these practical, real-world mini-projects, I aim to master idiomatic Go, standard library packages, and project architecture.

## 🚀 Learning Path & Progress

- [ ] **01- Basics:** Variable declarations, data types, loops (`for`), and conditional statements (`if`/`else`).
- [ ] **02- Gophercises Projects:**
  - [ ] **Quiz Game:** CSV parsing, goroutines, and timers.
  - [ ] **URL Shortener:** HTTP handler, YAML/JSON parsing.
  - [ ] **Choose Your Own Adventure:** HTML templates, web server, and JSON parsing.
- [ ] **03- Advanced Concurrency:** Mastering Go's strongest suit—Goroutines and Channels.
